Ensure you add .env.local file with credentials:
```
AWS_REGION=
AWS_ACCESS_KEY_ID=
AWS_SECRET_ACCESS_KEY=
DYNAMO_TABLE_NAME=
DYNAMO_TABLE_NAME_M=
S3_BUCKET=
S3_BUCKET_M=
S3_BUCKET_C=

GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=

NEXTAUTH_SECRET=
NEXTAUTH_URL=http://localhost:3000
NEXT_PUBLIC_BASE_URL=http://localhost:3000
```

How to run:
```
cd coe
npm run dev
```
